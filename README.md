# JavaScript Interactive Webpage - Event Listeners

## Overview
This project was developed for **Web Technologies Laboratory - Experiment 8**. It demonstrates how JavaScript event listeners can be used to make a webpage interactive through user actions.

## Features
- Change button color using a click event
- Update webpage heading and text dynamically
- Interactive box that changes color, rotates, and scales on hover
- Demonstrates DOM manipulation
- Uses the `this` keyword to modify the clicked element

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Learning Outcomes
Through this project, I learned how to:
- Attach event listeners using `addEventListener()`
- Handle user interactions such as clicks and mouse events
- Manipulate HTML elements using the DOM
- Use the `this` keyword inside event handlers
- Create interactive and responsive web pages

## Reflection
This experiment helped me understand how JavaScript event listeners make webpages interactive. I learned how different events such as click and mouseover can dynamically change webpage elements. I also understood the purpose of the `this` keyword, which refers to the element that triggered the event, allowing me to modify that specific element directly. Designing my own interactive webpage improved my understanding of event handling and DOM manipulation.
